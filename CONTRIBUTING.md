# Contributing to SkinBridge

Thank you for your interest in contributing to SkinBridge.

SkinBridge is a web-based tool that converts Minecraft skins (64x64) into editable 3D models and Roblox clothing templates.

## Getting Started

1. Fork the repository.
2. Clone your fork.

```bash
git clone https://github.com/your-username/minecraft-to-roblox-clothing-exporter.git
```

3. Install dependencies.

```bash
npm install
```

4. Start the development server.

```bash
npm run dev
```

## Development Guidelines

* Keep code clean and maintainable.
* Follow the existing project structure.
* Use meaningful commit messages.
* Test your changes before submitting a pull request.
* Document significant changes when necessary.

## Pull Requests

Before opening a pull request:

* Ensure the project builds successfully.
* Verify that existing functionality is not broken.
* Provide a clear description of your changes.
* Reference related issues when applicable.

## Reporting Bugs

If you find a bug, please open an issue and include:

* A clear description of the problem.
* Steps to reproduce it.
* Expected behavior.
* Screenshots or logs if available.

## Feature Requests

Feature requests are welcome. Please open an issue describing:

* The problem you are trying to solve.
* Your proposed solution.
* Any additional context or examples.
